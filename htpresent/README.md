# htpresent (HyperText Presentation)

simple script for making presentations

## how-to:
* set element coordinates
```html
<h1 coords="500 200">h1!</h1>
```

* set background image
```html
<img src="https://133.7.42.69/mybg.png" class="full_img">
```
